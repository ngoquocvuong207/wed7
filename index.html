Tổng quan nhập môn web

Hệ thống kiến thức nền tảng về phát triển ứng dụng Web.

  • internet: mạng máy tính toàn cầu kết nối với các thiết bị.
  • web: hệ thống thông tin trên internet cho phép truy cập qua URL.
  • website: tập hợp các trang web con liên kết với nhau.
  • trang web: một trang nội dung đơn lẻ hiển thị trên trình duyệt thuộc một website.
  • browser: phần mềm hiển thị trang web như chrome và cốc cốc.
  • server: nơi lưu trữ dữ liệu và xử lý dữ liệu của website để gửi về browser.
